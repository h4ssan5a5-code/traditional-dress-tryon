# Pakistani Traditional Dress Virtual Try-On 👕✨

AI-powered virtual try-on backend built with FastAPI that allows users to upload a person image and clothing image to generate realistic Pakistani traditional dress try-on results.

## Features

- FastAPI backend
- AI virtual try-on pipeline
- Upload person + clothing images
- Generate realistic try-on outputs
- REST API endpoints
- Result image retrieval
- Async image processing
- Production-ready structure
